* CSS 활용하기 1. Inline(인라인)

html 요소의 style에 CSS를 넣기 (좋은 방법이 아니다.)

* CSS 활용하기 2. Embedding(내부참조)

html 내부에 CSS 포함시키기 (h1 내부에 color 선언 후 작동)

* CSS 활용하기 3. link file(외부참조)

외부에 있는 CSS파일 불러오기
